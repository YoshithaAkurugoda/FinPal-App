# FinPal — Intelligent Personal Finance Companion

## Project Overview

FinPal is a mobile-based personal finance management application designed to help users track their spending, manage budgets, set savings goals, and receive intelligent financial insights.
The system integrates backend financial processing with an AI service that analyses user spending behaviour and generates personalised recommendations.

## Key Features

* Secure user authentication and account management
* Manual and automatic transaction recording
* Budget planning and utilisation tracking
* Savings goal creation and progress monitoring
* AI-powered financial insights and chat assistance
* Weekly and monthly financial summary reports
* Receipt upload and SMS-based transaction extraction
* Balance reconciliation and spending analysis

## System Architecture

The application follows a layered architecture consisting of a mobile frontend interface, backend service layer, database storage layer, and an integrated AI analysis service.

## Technologies Used

* Mobile Application Development Framework
* Backend API Services
* Relational Database System
* Artificial Intelligence Integration
* Cloud File Storage Handling
